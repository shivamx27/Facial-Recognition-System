# Facial Recognition System

A Face Recognition System built using Python and OpenCV.

## Features
- Face Detection
- Face Data Collection
- Model Training
- Face Recognition

## Technologies Used
- Python
- OpenCV
- Haar Cascade Classifier

## Files
- datacollect.py - Collects face images
- training.py - Trains the model
- testmodel.py - Tests face recognition
- haarcascade_frontalface_default.xml - Face detection model
